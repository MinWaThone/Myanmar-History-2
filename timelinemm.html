ဟံသာဝတီ နိုင်ငံတော်အလံ

(၁၂၈၇ - ၁၅၅၂)

ကုန်းဘောင်မင်းဆက်မြန်မာနိုင်ငံအလံ

(၁၇၅၂ - ၁၈၈၅)

ဗြိတိသျှဘားမား၏ ကိုလိုနီခေတ်အလံ

( ၁၈၈၆ – ၁၉၄၂ ; ၁၉၄၅ – ၄ ဇန်နဝါရီလ ၁၉၄၈)

ဗြိတိသျှဘားမားကိုလိုနီခေတ်အလံ

( ၆ ဖေဖော်ဝါရီ ၁၉၃၉ – ၃၀ မတ်လ ၁၉၄၁;[၃၅] ၁၉၄၅ – ၃ ဇန်နဝါရီလ ၁၉၄၈ )

ဗြိတိသျှဘားမား၏ ယာယီကိုလိုနီအလံ

( ၃၀ မတ်လ ၁၉၄၁ – ၁၉၄၂ )

ဂျပန်စစ်အုပ်ချုပ်ရေး၏ မြန်မာနိုင်ငံ ယာယီအလံ

( ၁၉၄၂ – ၁၉၄၃ )

ယာယီနိုင်ငံတော်အလံ

( ၁ သြဂုတ်လ ၁၉၄၃[၃၅] - ၁၉၄၅ )

မြန်မာနိုင်ငံအလံ၏ ပုံစံကွဲပုံစံဒီဇိုင်း

( ၁၉၄၃ - ၁၉၄၅)

ပြည်ထောင်စုမြန်မာနိုင်ငံတော်အလံ

( ၄ ဇန်နဝါရီလ ၁၉၄၈ - ၂ ဇန်နဝါရီလ ၁၉၇၄ )

ပြည်ထောင်စု ဆိုရှယ်လစ်သမ္မတမြန်မာနိုင်ငံတော်အလံ

( ၃ ဇန်နဝါရီလ ၁၉၇၄ - ၂၁ အောက်တိုဘာလ ၂၀၁၀ )

ပြည်ထောင်စုသမ္မတမြန်မာနိုင်ငံတော်၏ နိုင်ငံတော်အလံ
